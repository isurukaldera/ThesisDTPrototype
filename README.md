## üèóÔ∏è Prototype Overview

This prototype demonstrates a **privacy-conscious Digital Twin** that bridges physical retail operations with digital intelligence through:

### Core Components
- **3D Virtual Environment** (Unity): Real-time store and warehouse visualization
- **AI Forecasting Engine**: Ensemble model (SMA + WMA + Linear Regression)
- **Heatmap Analytics**: Sales performance visualization
- **Inventory Management**: Automated alerts and restocking logic
- **Simulated Database**: 350+ products across 10 categories

### Key Research Contributions
- GDPR-compliant approach focusing on product movement rather than customer tracking
- Integration of RFID-inspired logic with simulated POS data
- Proof-of-concept for spatial anchoring and real-time product visualization
- Validation of ensemble forecasting models in retail context
